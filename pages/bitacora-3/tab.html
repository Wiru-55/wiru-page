ETIQUETA: TABLE

IDEAS CLAVE

  • Elementos de la tabla:

    • Thead: Encabezado

    • th: Celdas de encabezado

    • Tbody: Cuerpo

    • Tr: renglones

    • Td: Celdas

    • Caption: Insercion de titulos

    • Tfoot: Pie de Pagina

NOTAS DE LA CLASE

Las tablas HTML permite organizar los datos en filas y columnas. Una columna consta de una tabla que contiene filas y columnas en celdas.

Cada tabla creada tiene su ancho y altura de manera automatica, es decir, tiene un acomodo dependiendo del contenido que se le añade.

Una tabla se define como un conjunto estructurado de datos distribuidos en filas y columnas. Permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algun tipo de conexión.

Una tabla se conforma en HTML como:

  • Thead: siendo el encabezado de la tabla
  • Th: las celdas del encabezado
  • Tbody: Cuerpo de la tabla
  • Tr: los renglones
  • Td: celdas de los datos
  • Caption: insertar titulos
  • Tfoot: El pie de pagina

Tambien dentro de la etiqueta Table se permite ajustar atributos para poder ocupar tanto el tamaño de 2 celdas, es decir, puede ser usado para unir/fusionar celdas
de una tabla.
Siendo estos atributos:
  • Rowspan: el indicador de el numero de filas que ocupara una celda
    < td rowspan= 3 >: la celda ocupara el alto de 3 celdas normales
  • Colspan: indicador del numero de columnas que ocupara la celdas
    < td colspan=2 >: refiriendose que la celda actual se extiende en ancho de dos celdas.

RESUMEN

Con esta etiqueta, nos permite crear varias tablas necesarias para reunir o apilar y mostrar los datos obtenidos de alguna manera para ya sea una investigacion como asi valores o datos importantes registrados.
