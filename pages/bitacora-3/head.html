HEADER

IDEAS CLAVE

    Elementos

    • Logotipo:
      sirve como guía visual para el usuario y permite reforzar la imagen de la marca

    • Nav:
      Elemento Opcional, navegar entre secciones

    • Buscador:
      Elemento util para ubicar cosas en la pagina de manera rapida y eficaz

    • Datos de Contacto:
      Se puede incluir telefonos o email.

    • Redes Sociales:
      Canal complementario para comunicar un mensaje.

    • Tienda Online: si es una tienda Online, se debe incluir en el header.

    • Botones Destacados:
      Puedes incluirlos para enfatizar un servicio.

NOTAS DE LA CLASE

El objetivo es aprender a identificar los elementos que pueden estar en un Header. Es una de las mas importantes en una pagina web, es el elemento que los usuarios visualizan en primer lugar cuando llegan al proyecto.

La cabecera debe ser ordenada y limpia para que pueda ser visualizada con un solo vistazo y no dé lugar a confusión. Debemos tener en cuenta que el header es un espacio genérico que tendrá el mismo diseño en todas las páginas web.

Lo primero que se debe hacer un listado con los elementos que va incluir en el header de tu web de forma obligatoria.
Los elementos pueden formar parte del header, van a depender de los objetivos y del tipo web.

  • Logotipo
  • Menu
  • Buscador
  • Datos de Contacto
  • Redes Sociales
  • Carritos de Compras
  • Botones
  • Selector de Idioma
  • Formulario de Suscripcion, etc

RESUMEN

El header de una página web es esencial ya que es lo primero que ven los usuarios al entrar al sitio. Debe ser ordenado y claro, conteniendo elementos como el logotipo, menú, buscador, datos de contacto, redes sociales, carrito de compras, botones destacados, selector de idioma o país, formulario de suscripción, entre otros.
Cada elemento tiene su función específica y su ubicación puede variar según el diseño y la estructura del sitio. Es importante evitar saturar el header con demasiada información y mantener el equilibrio visual.
