MODULO GRID

IDEAS CLAVE

  • Grid permite diseñar filas y columnas, a diferencia de flexbox que es unidireccional.

  • Al definir un contenedor con display:grid, todos los hijos de este se convertiran en cuadriculas.

  • Gap simplifica el espacio entre filas y columnas.

  • Fr asigna fracciones del espacio disponible

  • grid-template-areas permite nombrar y organizar areas especificas dentro del grid para mayor flexibilidad en el diseño.

NOTAS DE LA CLASE

Comunmente se ha utilizado varias estatregias de CSS para diseñar paginas web. Primero se usaba table, luego fue float-position y display, pero de todos modos estos metodos omiten funciones importantes como lo es: centrado vertical.

CSS Grid Layout

  • Grid es un sistema de diseño bidimensional basado en cuadriculas que en comparacion con cualquier otro sistema de diseño web del pasado, cambia por completo la forma en que se dimensiona las interfaces de usuario.
  • Fue creado especificamente para la resolucion de problemas de diseño y que a diferencia de flexbox (el cual es de flujo unidireccional), Grid permite definir un esquema de cuadricula.
Soporte
  • Apartir del 2017, se empezó a ofrecer soporte nativo sin prefijos para CSS GRID en todos los navegadores:
    • Chrome
    • Firefox
    • Safari
    • Opera
    • Internet Explorer
Display: Grid
Para comenzar, se debe definir un elemento contenedor como una cuadricula con la expresion: Display: Grid;
El cual todos los hijos directos del contenedor de la cuadricula se convierten automaticamente en elementos del grid.
  • Grid-column - las lineas verticales son las columnas
  • Grid-rows - las lineas horizontales son los rows
Propiedades
  • Gap
    La propiedad gap permite resumir las propeidades de la columna-gap y row-gap
  • Fr
    Es una medida de CSS Grid, es la abreviacion de "fraccion" y es utilizada para especificar fracciones del espacio disponible entre una cuadricula CSS, util para cuando se requiere el espacio restante despues de colocar elementos en la cuadricula.
  • template-areas
    Permite definir un esquema de cuadricula utilizando nombres de areas personalizadas en lugar de especificar las lineas de la cuadricula con grid-template-rows y grid-template-columns.

RESUMEN

Gracias a este modulo, nos permite organizar de una manera precisa y adaptable en forma bidimensional en los elemtnos en una pagina web. Ya que fue diseñado especificamente para la resolucion de desafios de diseño, dando como resultado una facildiad de creaciones de de estructuras complejas y responsivas.
