  • Existen varias valores o maneras de reproducir una animacion con Animation-Direction:
    • Normal: Se reproduce de manera hacia adelante.
    • Reverse: Se reproduce de manera invertida.
    • Alternate: Primero hacia adelante y luego hacia atras.
    • Alternate-reverse: similar que alternate pero de manera invertida.

NOTAS DE LA CLASE

- CSS3 permite animar la mayoria de los elementos HTML sin usar JavaScript.
Cuando se especifica estilos CSS dentro de la regla de keyframes, la animacion cambiara del estilo actual al estilo nuevo.

- Para que una animacion funcione se debe vincular la animacion a un elemento Div y asignarle en CSS estos valores:
Div { animation-name: animacion-1} y
@keyframes animacion-1{
From {background-color: red}
to {background-color: yellow} }

- Existen varias propiedades para manejar la animacion:

  • Animation-duration: que especifica cuando tiempo debe transcurrir una animacion para completar un ciclo.

  • Animation-timing-function: especificando la velocidad de una animacion con una corvatura.

  • Animation-direction: que permite especificar si una animacion debe reproducirse de manera hacia adelante o atras o en ciclo alterno.
