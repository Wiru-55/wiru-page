NOTAS DE LA CLASE

HTML5 presenta nuevos elementos utiles para creacion de paginas web actuales, reduciendo los pasos necesarios para crear uno. Estos nuevos elementos se clasifican en 2 grupos:

  • Elementos de Nuevas Funcionalidades: donde estas sirven para extender el HTML, dando soportes a temas como es el video, el sonido, lienzos donde dibujar, etc.

  • Elementos de Web Semantica: donde sus elementos no sirven para nuevas funcionalides, si no para componer sitios indicando que son los bloques de codigo web, en lugar de representar, es decir, describen su significado tanto para el navegador como para el desarrollador.
    • < Main>
    • < Header>
    • < Nav>
    • < Section>
    • < Article>
    • < Footer>

RESUMEN

En terminos mas simples de HTML5, la razon de porque se implemento este nuevo sistema fue porque anteriormente los desarroladores usaban los nombres utilizando los ID/class, esto implicaba que los buscadores de los navegadores batallaran para en identificar el contenido correcto de una pagina web.
Por esta razon, se crearon estos elementos para reducir la carga de busqueda y tener una mayor facilidad de escritura de codigo.
