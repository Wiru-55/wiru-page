NOTAS DE LA CLASE

- Elemento Form: Form define un formulario que se utiliza para recoipilar imformacion que el usuario escribe.

- Elemento Input: Es el mas importante en un formulario. Puede ser mostrado de diferentes formas dependiendo del atributo type.

  • Input no tiene etiqueta de cierre.
  • El atributo "PlaceHolder" es para agregar texto al cuadro.
  • Con el elemento "label" se utiliza para ligar elemento input y poder activar el cuadro de texto, tambien como agregar texto que referencie al input.
  • La diferencia entre input y text area es que el text area pone un limite de demostracion de texto para luego agregar un scroll de navegacion el cuadro de texto y el input es un region sin limite
- Entre los distintos tipos de entrada para el elemento Input son:
  • SUBMIT: Es un boton para enviar informacion escrita en cuadros de texto hacia el servidor

  • PASSWORD: Similar a un cuadro de texto, permite comparar una variable de texto con el ingresado en el servidor, pero que sin que se vea el texto en el cuadro.

  • RESET: Crea un boton de reset que limpiea todos los campos a los valores iniciales.

  • RADIO: Crea un boton circular de seleccion.

  • CHECKBOX: Similar a Radio, pero de forma cuadrada y de seleccion multiple.

  • BUTTON: Ejecuta un evento HTML.

  • FILE: Permite cargar una imagen o archvio al servidor.

  • DATE: Es usado para campos que contiene una fecha.

  • TEL: Se utiliza para ingresar un numero telefonico

  • COLOR: permite que el usuario seleccione un color.

  • SELECT: Permite seleccionar varias opciones predeterminadas por el programador en un cuadro
