DISPLAYS

IDEAS CLAVE

    Block
  • Cualquier elemento lo tiene por predeterminado, siempre ocupara el ancho posible y hara saltos de renglon.

    • INLINE
  • Nunca comenzara en una nueva linea, ocupara el ancho que sea necesario y permite que otros elementos se agreguen a su lado.

    • INLINE-BLOCK
  • Establece un ancho y una altura dependiendo del elemento, tambien permite ajustarlo manualmente, ademas de permitir agregar paddin y margin.

    • NONE
  • Usado en JavaScrip permite ocultar elementos haciendolos que desaparezca su espaciado en la pagina web hasta que sea necesario.

    • TABLE
  • Permite asignar en "tablas" como las tablas tradicionales pero HTML.

NOTAS DE LA CLASE

- La propiedad display es una de las mas complejas, establece el tipo de la caja que genera cada elemento.

- Cada elemento HTML tiene un valor de visualiacion predeterminado segun el tipo de elemento que sea. El valor de visualizacion predetermiando para la mayoria de los elementos es BLOCK o INLINE.

BLOCK
Un elemento con display block siempre comienza en una linea y ocupa todo el ancho disponible (de derecha a izquierda tanto lo que sea posible).
Los elementos block hacen un tipo de salto de renglon al agregar otro elemenbto de su tipo.

    Ejemplos
  • DIV
  • P
  • Header
  • Section
INLINE
Un elemento inline no comienza en una linea y solo ocupa el ancho que sea necesario. Permite agregar mas elementos a lado de ellas y sin dejar que se pongan debajo de ellas.
  • -"Los valores de ancho y altura no permiten utilizarse en display inline, se deben agregar un display inline-block o block."
INLINE-BLOCK
Permite establecer un ancho y una altura en el elemento. Ademas permite los margin y los padding.
No agrega un salto de linea despues del elemento, permitiendo que un elemento nuevo pueda alinearse junto a otros.

NONE
Se usa comunmente con JavaScript para ocultar y mostrar elementos cuando el usuario realiza alguna accion como pulsar un boton o un enlace sin eliminarlo ni crearlos nuevamente.
Lo que hace el elemento no genere una caja. El resultado es que el elemento desaparece por completo de la pagina y no ocupa sitio.

TABLE
Permite asignar un comportamiento de tabla de elementos que haya dentro (andiados a la etiqueta donde esta colocando el table).
Dentro de un elemento marcado como display table, se pueden usar otra serie de valores de siplay que corresponden con los roles de distintas etiquetas HTML que se usan en las tablas tradicionales.
  • Display Table-ROW: actua como un elemento en fia, etiqueta "b"
  • Table-CELL: etiquetas que trabajan como una celda de tabla, etiqueta "id"

RESUMEN

Con la propiedad DISPLAY nos permite mucho en poder desarrollar y controlar la manera en que se estructura una pagina, permitiendo unir varios cuadros de texto o acomodando cuadros de imagen o texto, para un mejor vistazo de la pagina.

DISPLAY ademas de permitirnos manipular el espaciado entre renglones de un elemento HTML, nos deja organizar de manera mas controlada al punto que es una de las propiedades mas utiles y versatiles que se crearon para CSS, siendo una pieza fundamental o algo basico para crear paginas de hoy en dia.
