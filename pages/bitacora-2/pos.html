HTML5 CSS3 POSITION DISPLAYS ANIMATION REFERENCIA FORMULARIOS

POSITION

IDEAS CLAVE

  • Casi todas las propiedades de posicion se pueden manipular con top-bottom-right/left y z-index.

  • Position Static: como su nombre indica se mantiene en su posicion como un bloque y se acomoda dependiendo a la posicion del contenido de la pagina web de manera normal

  • Relative: al ser manipulado tratara de acomodarse al div mas cercano dependiendo si tiene una posicion diferente a Static.
  • Absolute: Al mover un elemento con esta propiedad, todos los elementos se solaparan en el espaciado que deje el elemento movido.

  • Fixed: No deja espaciado entre los elementos, es decir mueve todo para poscionarlo en un lugar mejor.

  • Sticky: Puedes indicarle de que posicion hasta que otra posicon puede moverse, es decir puedes delimitar un punto del elemento donde ya no aparezca mas en la pantalla.

NOTAS DE LA CLASE

La propiedad Position permite crear algun tipo de metodo de posicionamiento, teniendo 5 valores que afectan el posicionamiento de un elemento HTML

Cada elemento se posicionan usando propiedades como lo son: top, bottom, left, right y z-index, pero estas propiedades funcionan de manera diferente dependiendo del valor de posicionamiento.

Los elementos de position son:

  • Static
    - Este elemento viene por predefinido en todos elementos HTML, y en cuanto a su valor de posicionamiento no se ve afectado por las propiedades de top,bottom,left-right y z-index, ya que se maneja mediante el flujo normal de la pagina web.

  • Relative
    - Tanto el posicionamiento relativo desplaza un elemento respecto de su posicion original establecida mendiante el posicionamiento normal.

  • Absolute
    - Esta posicion se emplea para establecer de forma exacta la posicion en la que se muestra la caja de un elemento.
    Cuando un elemento se posiciona de forma absoluta, el resto de los elementos se ven afectados y ocupan el lugar original del elemento anterior solapandose entre si.

  • Fixed
    - Esta ubicado en relacion con la ventana grafica, lo que signicia que siempre permanece en el mismo lugar, incluso si la pagina se desplaza. Tantos las propiedades top-bottom-right/left.. se utilizan para reposicionar un elemento, en si esta propiedad no deja un espaciado original de un elemento movido.

  • Sticky
    - Un elemento con posicion Sticky permite posicionarse en funcion a la posicion del desplazamiento del usuario, alternando entre relativo y fijo, dependiendo de la posicion de desplazamiento.

RESUMEN

Con estas propiedades, se puden utilizar como para crear imagenes que se mantengan durante la travesia de la pagina web, como poner imagenes o informacion en ciertos lugares que normalmente no se dejaria poner.
