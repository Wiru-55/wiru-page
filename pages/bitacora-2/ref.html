HTML5 CSS3 POSITION DISPLAYS ANIMATION REFERENCIA FORMULARIOS

REFERENCIAS

IDEAS CLAVE

  • NOT() nos permite agregar excepciones a estilos con reglas agregadas a un elemento muy especifico.

  • Para referenciar con hijos de los elementos:

    • First-child: solo referencia al primer hijo.

    • Last-child: similar pero con el ultimo hijo.

    • Only-child: y este solamente al unico hijo que haya en un elemento.

  • Utilizando solamente los atributos para referenciar algun elemento, nos permite especificar de manera controlada:

    • $="mi": referenciar un atributo que comienze con "mi".

    • *="mi": similar pero referencia algo que termine con "mi".

    • ^="mi": Aquel que tenga un elemento relacionado a "mi".

NOTAS DE LA CLASE

- Tanto las referencias ID y Class nos permite identificar algun elemento en especico, la mayoria de las veces se resulta un poco complicado y tambien repetitivo tener que escribir lo mismo o crear varias variables para hacer diferentes estilos en css, gracias a una nueva implementacion de CSS3, se incorporó unas pseudo clases que permiten seleccionar elementos de manera mas especifica.

- Con las pseudoclasses podemos aprovechar esta organizacion y referenciar un elemento especifico sin importar cuanto conocemos sobre sus atributos y el valor de los mismos.
Utilizando la pseucoclase: nth-child() nos permite encontrar un hijo especifico de un elemento

  • p:nth-child(2) {background-color:#f08b5}
- Tambien existen otras pesuclases relacionados a esto ultimo, como por ejemplo:
  • First-child
  • Last-child
  • Only-child
- Otra referencia util es not() que permite declarar una excepcion, como ejemplo se puede poner que en un una regla de estilo creada cualquier elemento que contenga not pero los demas no, se aplicara a todos menos al que tenga la referencia NOT()

- Tambien gracias a CSS3, se implemento mas maneras hacer referencias, como por ejemplo, referenciar a elementos utilizando atributos de los mismos elementos sin tener que agregar Clases o ID´s en el documento CSS.
CSS3 permite combinar "=" con otros atributos y valores para hacer una seleccion mas especifica:
  • P[name $="mi"] {font-size: 20px}
    (Permite definir un atributo que empieze con "mi")

  • P[name *="mi"] {font-size: 20px}
    (Este asigna a cualquier elemento que tenga un valor final a "mi")

  • P[name ^="mi"] {font-size: 20px}
    (Y este asigna todo elemento que tenga un atributo id cuyo valor incluya el texto "mi")

RESUMEN

Con todas estas nuevas implementaciones de referencias, nos permite de manera muy especifica, como agregar elementos sin tener que agregar mas "Clases" o "ID´s" a nuestra hoja estilos.

Estos nos puede servir, para crear ciertos elementos vistosos o ejemplos que necesitan algo visual pero sin afectar a otros elementos necesarios.
