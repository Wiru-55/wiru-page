EVENTOS

IDEAS CLAVE


  • Los eventos son cuando se interactua de manera simple con los elementos HTML
    • onclick
    • ondbclick
    • onmouseover
    • onmouseout
    • onload
    • onkeydown
    • onscroll

  • Tanto add como remove permite que podamos agregar o quitar eventos que puedan sernos utiles para tener una pagina mucho mas dinamico.

NOTAS DE LA CLASE

HTML DOM permite ejecutar codigo cuando se ocurre algun evento. El navegador genera eventos cuando "suceden cosas a elementos HTML", ya sean

  • Al hacer clic con el mouse - onclick
  • Al hacer doble clic con el mouse - ondbclick
  • Colocar el mouse sobre un elemento - onmouseover
  • Alejarlo de un elemento - onmouseout
  • Cuando se carga una pagina web - onload
  • Presionar una tecla - onkeydown
  • Con el scroll en la pagina - onscroll
EventListener
  • El metodo para agregar un evento es con - addEventListener() que adjunta un controlador de evento al elemento especifico.
  • Con la sintaxis seria:
    "X Elemento".addEventListener(evento, funcion)
  • Con document.addEventListener() se usa para adjuntar un controlador de eventos al documento HTML
RemoveEventListener
  • Siminar a add solo que con eliminar eventos agregados con el metodo removeEventListener()
  • Para eliminar controladores de eventos, la funcion especificada con el metodo addEventListener() debe ser una funcion externa.
