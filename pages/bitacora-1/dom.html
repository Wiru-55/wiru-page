NOTAS DE LA CLASE

El modelo de objetos de documentos (DOM), es una plataforma y una interfaz de lenguaje neutral que permite que los programas y scripts puedan acceder y actualizarse de manera dinamica al contenido, estructura y estilo de un documento.

Con el DOM, los programadores pueden construir documentos, navegar por su estructura y a√±adir, modificar o eliminar algun elemento o contenido.

HTML DOM es un modelo de objetos estandar y una interfaz de programacion para HTML que define

  • Elementos HTML como objetos.
  • Propiedades de todos los elementos HTML.
  • Metodos para acceder a todos los elementos.
  • Eventos para todos los elementos HTML.

  • En palabras mas sencillas, es un estandar sobre como se obtiene, cambia, o elimina elementos HTML.
Con la "Conversion de Tokens" que permite que un navegador convierte cadenas de caracteres en tokens diferentes, especificados por el estandar de W3C HTML5, ejemplos: < html > y < body >, etc.

De la manera en que como se construye el DOM, los objetos creados se vinculan en una estructura de datos en forma de arbol que muestra las relaciones entre objetos primarios y secundarios definidas en el HTML original.

RESUMEN

De manera resumida, DOM es una interfaz que permite a los programadores generar, manipular, eliminar elementos o contenido, ya que establece propiedades y metodos para poder usarlos.

Asi tambien, este se organiza de una manera mas ramificada utilizando datos y a su vez elementos, quienes son vinculados entre si para poder tener una relacion entre elementos primarios y secundarios.
