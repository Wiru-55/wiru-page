JS CONCEPTOS BASICOS

IDEAS CLAVE


  • JS permite realizar funciones mas complejas dentro HTML.

  • String - Valor en cadena con mas de 0 caracteres UNICODE.

  • Numero - Desde JS los valores no hay distincion entre enteros y puntos flotantes.

  • Boolean - Solo tiene 2 valores, siendo true o false.

NOTAS DE LA CLASE

JavaScript es un conjunto de reglas que permiten realizar una tarea o calcular un valor, para usar una función debe declararse en algún y se ejecuta cuando algo lo llama.

La sintaxis es: Function name([param...])

  • El codigo dentro de la funcion se ejecutara cuando "algo" invoque o llame a la funcion:
    • Cuando ocurre un evento (cuando un usuario hace click un boton) < button type="button" onclick="miFuncion"> Click aqui < / button>
    • Cuando se invoca desde el codigo de JavaScript: Document.getElementById("Demo").innerHTML="La temperatura es + toCelcius(77)+ "Celcius"

Tipos de datos:

  • String - Se refiere a un valor que esta hecho en manera de cadena de cero o mas caracteres Unicode.
  • Numero - Desde JavaScript, no se puede dinstinguir entre valores enteros y puntos flotantes.
  • Boolean - Algo similar a tener cadenas y numeros pero este solo puede tener tan solo 2 numeros, siendo literalmente true y false.

RESUMEN

En palabras mas sencillas, javascript funciona como un sistema diseñado en codigos para realizar tareas mas especificas y complejas. Las funciones de javascript pueden declarar y ejecutar estas tareas solamente cuando son llamadas.
Tambien JavaScript en sus funciones permite utilizar datos en valores ya sean: numericos, en strings o booleanos.
