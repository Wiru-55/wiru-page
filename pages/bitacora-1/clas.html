CLASS LIST

IDEAS CLAVE


  • Add permite agregar una nombre de una propiedad. Si ya existe uno, no se creará.

  • Remove como su nombre indica, permte eliminar/borrar uno o mas nombres de una clase.

  • Toggle alterna entre nombres de clase de cada elemento.

NOTAS DE LA CLASE

La propiedad ClassList devuelve los nombres de clase de un elemento. Esta propiedad es util para agregar, eliminar y alternar clases CSS en un elemento.

La propiedad classList es de solo lectura, sin embargo, puede modificarla utilizando los metodos add(), remove() y toggle().

  • classList.add() - es el metodo que agrega uno o mas nombres de clase a un elemento. Si la clase especificada ya existe, la clase no se agregará.

  • classList.remove() - La propiedad classlist mas el metodo remove elimina uno o mas nombres de clase de un elemento. Eliminar una clase que existe, no arroja un error.

  • classList.toggle() - El metodo alterna entre un nombre de clase para un elemento. El primer parametro elimina la clase especificada de un elemento y devuelve falso.
    • Si no existe, se agregará un elemento y valor de retorno es verdadero.

RESUMEN

Esta propiedad ayuda a poder cambiar los estilos de algun elemento de una manera mas rapida, ya que puedes agregar, quitar o cambiar clases de un CSS para modificar en como se verá o como se comportará sin realizar tanto codigo.
